## 文档结构

```
[封面]          标题 + 作者 + 日期
[目录]          四部分 + 14章编号标题
[前言]          约300字，说明本书目的与结构逻辑
──────────────────────────────────────
[第一部分]       认知基础（3章）
[第二部分]       人机关系（4章）
[第三部分]       创造与边界（3章）
[第四部分]       架构与商业（4章）
──────────────────────────────────────
[后记]          约200字，收敛全文核心观点
```

## 内容编辑规范

### 删除项
- Frontmatter 元信息（tags、date、description）
- 博客特有标记（原文链接、平台标注）
- 文末 AI 辅助说明（"以下由 AI 辅助生成"等）
- 文末 AI 对话评注段落（如 #2 末尾的 AI 评注）
- 原文末尾的 Reference 区块（如 #1 文末的参考链接列表）

### 保留项
- 所有正文章节结构
- 引用块、列表、表格
- 代码示例和 ASCII 图表
- 附录中仍有阅读价值的补充说明

### 风格统一
- 引用块统一用灰色底纹
- 标题层级统一为：部分标题 > 章节标题 > 子标题
- 列表符号全书统一（正文用 `·`，代码/结构用 `-`）
- 术语统一（如 LLM/大模型/模型，Agent/智能体）

## 排版规范

**页面**：A4，页边距上下 2.5cm，左右 3cm

**字体层级**：
```
封面标题    黑体 28pt，居中
部分标题    黑体 18pt，居中，篇序 + 名称
章节标题    宋体 16pt，左对齐，编号 + 标题
正文        宋体 11pt，行间距 1.8
引用块      宋体 10.5pt，左缩进 2em，#f5f5f5 底纹
代码块      等宽 10pt，#f5f5f5 底纹
页眉        书名左，页码右（Times New Roman 9pt）
```

**水印**：斜体 45°，右下角，内容「Kingson's AI Notes · 2026」，颜色 #cccccc

## 生成工具链

- **中间格式**：HTML（便于 CSS 控制）
- **渲染引擎**：weasyprint（pip install weasyprint）
- **水印**：CSS `@page` 规则实现
- **字体**：系统宋体/黑体（macOS 丽黑/苹方/华文）

## 生成流程

1. 提取 14 篇 Markdown，清理元信息，格式统一
2. 拼接为单本 HTML（加封面、目录、前言、后记）
3. weasyprint 渲染为 PDF
4. 输出：`output/AI思考小册子-2026.pdf`
